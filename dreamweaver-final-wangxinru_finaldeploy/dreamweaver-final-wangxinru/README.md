An interactive art-science project exploring global dream patterns.

## Features
- Dream interpretation (API Ninjas)
- Emotion analysis (HuggingFace)
- Visual dream imagery (Unsplash)
- Global dream trends (PyTrends)
- Cultural interpretation (Wikipedia)
- AI narrative generation (HuggingFace GPT2)
- Dream soundscape (Pixabay)
- Save dream card

## Setup
Create `.streamlit/secrets.toml` with your API keys:

```toml
ninjas_key = "your_api_ninjas_key"
hf_token = "your_huggingface_token"
unsplash_key = "your_unsplash_key"
pixabay_key = "your_pixabay_key"
```

Install dependencies and run:

```bash
pip install -r requirements.txt
streamlit run app.py
```
